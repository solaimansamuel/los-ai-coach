# 💎 LOS AI Coach (Hebrews 12 System)

## 🚀 One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

---

## 🧠 What This Does

- Reads Daily Events from Notion
- Sends to AI (ChatGPT)
- Writes coaching back to Notion
- Runs automatically every 10 minutes

---

## 🔧 Setup

Add environment variables in Vercel:

- NOTION_API_KEY
- OPENAI_API_KEY
- DAILY_DB_ID
- COACH_DB_ID
